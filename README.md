## 概要

ハッカソン用に cesium を使ったサンプルアプリを作成.
plateau というサービスで３ D データがあったので使用してみた.

## How to start

```
npm i
npm start
```

## 環境

- build: vite
- lib: cesium, resium, react, ts
- deploy: firebase

## デプロイ

```
npm run deploy
```

deploy target URL: <https://plateau-vite.web.app/>
